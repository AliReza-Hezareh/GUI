# Brewscape - Enkätfrågor med motivering

| Nummer | Fråga | Motivering |
|---|---|---|
| 1 | Vad var ditt första intryck av Brewscape? | Första intrycket visar om sidan direkt känns seriös, tydlig och trygg för en ny användare. |
| 2 | Förstod du snabbt vad sidan säljer? | Om användaren inte förstår syftet snabbt finns risk att personen lämnar sidan. |
| 3 | Hur lätt var det att hitta produktsidan? | Navigation är kritisk. Användaren måste enkelt hitta till produkterna. |
| 4 | Hur lätt var det att hitta en relevant produkt? | Testar om produktlista, sök och filter hjälper användaren att hitta rätt. |
| 5 | Fanns det någon text som var svår att läsa eller förstå? | Fångar problem med språk, svenska tecken, otydliga labels och läsbarhet. |
| 6 | Hur tydlig var informationen på produktsidan? | Produktinformation måste vara tydlig för att användaren ska kunna fatta köpbeslut. |
| 7 | Kändes knappar och länkar enkla att förstå? | Testar om användaren förstår vad som händer vid klick, till exempel kundvagn eller navigation. |
| 8 | Om du testade på mobil: fungerade layouten bra? | Många använder mobil. Layouten måste fungera på liten skärm utan overflow eller trasig meny. |
| 9 | Fick du tydlig återkoppling när du sökte, klickade eller fyllde i något? | Användaren behöver feedback för att förstå om en handling lyckades eller misslyckades. |
| 10 | Vad var det största problemet du hittade? | Öppen fråga som fångar problem som de andra frågorna kan missa. |
